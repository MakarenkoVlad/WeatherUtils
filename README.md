# WeatherUtils

## My second project

This application gives forecast up to 5 days and notifies you of the weather you chose.

Used technologies:

* Architecture components
  * View model
  * Room
  * Live data
  * Work manager
* Coroutines
* Retrofit

*Built according to MVVM design pattern*
